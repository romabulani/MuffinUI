Badges

Badges can be combined with Avatars, they are useful for status of user like busy, offline and away. They can also be used on icons for notifications.

Badges for Image Avatars

The first variant of Badge is for Round Image Avatars.For Round Avatars, class name badge-round-avatar needs to be added in span tag. As per the avatar size selected, the badge size differs, the sizes for badges are badge-lg, badge-md, badge-sm, badge-xs. Based on the status of the user, the different choices are badge-busy, badge-away and badge-online.

The second variant is for square avatars, only difference is that, you have to add class as badge-square-avatar

Badges for Icons

For Icon Badges, the three classes are to be added in span tag badge-icon, badge-number and badge-right
