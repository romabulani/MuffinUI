Buttons

Buttons are useful in performing various actions on a single click. Different variants of buttons are present in Muffin UI. Take a look

Default Buttons

The default buttons are available in different variants. The btn class needs to be added in button tag. Along with that classes such as btn-primary, btn-secondary, btn-default, btn-success, btn-error and btn-disable can be added as per the user's requirement

Outline Buttons

For outline buttons, the classes such as btn-outline-primary, btn-outline-secondary, btn-outline-default, btn-outline-success, btn-outline-error and btn-outline-disable can be added

Link Buttons

The links can also be wrapped in buttons by giving class name as btn-link to a tag. There are 3 variations primary, secondary and default. For primary the class btn-link-primary can be given.

Buttons With Icons

The class btn-icon needs to be added along with other classes.

Floating Action Buttons

These can be used by providing the class name as btn-float
