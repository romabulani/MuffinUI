Muffin UI

Make your UI clean and responsive

Muffin Component Library is the user-friendly and lightweight component library. It helps you to design your responsive website quickly just by importing it in your project.

Why Muffin UI?

Responsiveness
You dont need to set the reponsive units by yourself. We encourage consistency across platforms, environments, and screen sizes by using uniform elements and spacing.
Lightweight
Whole library is made using Vanilla CSS3, which makes it lightweight. This helps in loading the website faster avoiding the user to wait for long time.
User Friendly
The names for CSS Classes are quite easy, keeping in mind the component's name, appearance and its use. The code snippets are provided to help the user.
